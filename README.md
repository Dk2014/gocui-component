# gocui-component
This is unofficial [gocui](https://github.com/jroimartin/gocui) component.  
Form and others can be created easily.

# Features
you can easily create the following components.

- Form
- InputField
- Button
- Dialog
- Table
- Radio
- CheckBox
- Select
